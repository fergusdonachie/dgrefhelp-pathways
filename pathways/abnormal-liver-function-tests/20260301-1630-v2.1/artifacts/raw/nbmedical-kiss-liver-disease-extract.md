## Practical points captured

- Stepwise approach:
  1. history and examination
  2. initial tests with clear indication
  3. pattern-specific work-up
  4. action/escalation by risk

- Bilirubin branch:
  - isolated unconjugated bilirubin with no anaemia -> likely Gilbert pattern
  - unconjugated bilirubin with anaemia -> investigate haemolysis (reticulocytes/LDH etc.)

- ALT/AST branch:
  - broaden work-up with AST, GGT, FBC, ultrasound and liver aetiology screen
  - if steatosis present, classify likely MASLD/MetALD/alcohol-related disease by risk context
  - use non-invasive fibrosis scoring (e.g., FIB-4/NAFLD score pathways) to triage primary care vs specialist pathway

- Cholestatic branch (ALP/GGT predominant):
  - evaluate cholestatic causes with ultrasound + aetiology testing
  - check vitamin D where ALP is raised (to avoid misattribution)

- Synthetic failure branch:
  - low albumin + raised INR should trigger urgent referral and urgent imaging while awaiting specialist review
